1、框模型

  框模型（Box Model 也以为“盒子模型”），是CSS3中一个非常重要的概念，也是比较抽象的一个概念。文档树中的每一个元素都是以一个矩形框的形式存在的，这些框影响了元素与元素之间的距离，元素内容的位置等；而浏览器根据视觉格式化模型来将这些框布局成访问者看到的样子。这个框模型规定了处理元素内容、内边距、边框和外边距的方式。如下图是一个框模型的示意图：

  从上图我们可以看出框模型类似于一个盒子，最里面的部分为元素内容，它有宽度和高度。而在往外有一条和黑色的边框，在边框与元素内容之间的称为padding（内边距），而在往外，就是最外面的那条虚线，虚线与边框之间的部分称为margin（外边距）。外边距默认是透明的，因此它不会遮挡其后面的任何元素。

  1.1、外边距（margin）

  外边距就是元素与元素之间的距离，它主要由四部分组成：margin-top、margin-right、margin-left和margin-bottom。下面以margin-top为例：

margin top:<length>|auto|;
◆auto:默认的边距（不管是那一种边距，其默认边距都为auto）

  外边距尽然有四个值，那么是不是就一定要一个一个去设置它呢？答案肯定不是啦！之前就已经讲过简写属性了，所以这里也可以使用简写属性，它分为以下几种情况：

  1、只设置一个外边距的值：当只给margin设置一个值的时候，它表示上、下、左、右的四个外边距的值都为那个数。
  2、设置两个外边距的值：当给margin设置两个外边距的值的时候，前面那个值表示上、下外边距的值，而后面那个数则表示左、右外边距的值；
  3、设置三个外边距的值：当给margin设置三个外边距的值时，前面的那个数表示上外边距的值，中间那个数表示左、右外边距的值，最后那个数表示下外边距的值；
  4、设置四个外边距的值时，它表示从顶端开始，按照顺时针的顺序，依次描述各边的值，即上、右、下、左。

  2.2、内边距（padding）

  内边距就是元素内容与对象边框之间的距离，它可通过padding属性进行设置，同样它也有四个属性值，设置的方法与前面一样（跟前面的一样，这里就不再叙述）。

  【注意】与外边距不同的是，关键字auto对padding不起作用，并且padding不能设置负值。

   2.3、边框

  边框的属性主要是通过边框颜色（border-color）、边框宽度（border-width）以及边框样式（border-style）来完成。

  1、border-color：该属性用来设置边框的颜色，同样也是使用之前所说的三种方法，它也有四个值，即上下左右，跟前面是一样的，这里也不再叙述。

  【注意】：color和style属性只有在设置了样式并且不为none、宽度值不为0的情况下，才会有效，否则不显示。

  2、border-width：该属性用来设置边框的宽度。它有以下四个值：

  medium：默认边框宽度

  thin：比默认边框窄

  thick：比默认边框粗

  length：指定具体的线条宽度

  3、border-style:该属性用来设置边框的样式，有以下几种样式：

outset 3D凸边
ridge 菱形边框
solid 实线
none 没有边框

2、布局常用属性

  1、浮动

  浮动（float）是css样式表中的浮动属性，用于给指定元素设置其浮动属性，改变元素的排列方式。它有三个属性值：

  ◆left：元素浮动在左侧

  ◆right：元素浮动在右侧

  ◆none：元素不浮动

  （1）、重点：浏览器认为浮动的元素是不会占用正常文档流的位置的，因此它可能会把后面的元素给遮挡住，所以会影响后面的元素。如下图所示：

  从上面的图中我们可以清楚的看到，当没有在三个元素（div）上设置浮动时，它遵循的是正常文档流的排序；而在把后面的两个元素设置了浮动（left）以后，中间的元素就会“消失”，其实它并没有消失，我们可以改变这个元素的宽度来查看它到底跑去哪了：

  可以看出这个元素其实跑到了第一个元素下面去了，这就说明当元素设置了浮动属性以后就不会遵循正常文档流的规则，这样也就影响了后面的元素。

  注意：

  1、浮动的元素可以设置width、height、margin、padding

  2、浮动的元素不区分行块元素，都能设宽高，并排显示，不独占一行

  3、浮动的元素不接受display转换显示模式，因为浮动的元素没有行内、块级元素之分

  4、浮动的元素如不设置宽高，会自动收缩宽度，会根据内容撑开

  5、浮动的元素不占标准流的位置

  6、浮动的元素不能设置margin:0 auto;居中

  2、定位

  在一个文本中，任何一个标签都被文本限制了自身的位置。但是可以通过定位（position）来改变自身的位置。定位就是把指定标签放在页面中的任何一个位置。该属性有以下几个值：

  ◆static：默认值。无特殊定位，遵循HTML定位规则，使用该属性时，设置top等都无效；

  ◆absolute：绝对定位。使用top等来定义绝对位置。使用该属性可以让元素浮于页面之上。

  ◆fixed：绝对定位，元素位置固定不变，也不随滚动条而改变位置。

  ◆relative：相对定位，元素遵循HTML定位规则，使用top等设置指定位置。

下面除了第一个以外的都给大家演示一遍：

h2
{
    position:absolute;
    left:100px;
    top:150px;
}

  用绝对定位,一个元素可以放在页面上的任何位置。标题下面放置距离左边的页面100 px和距离页面的顶部150 px的元素。

p.pos_fixed
{
    position:fixed;
    top:30px;
    right:5px;
}

  使用fixed绝对定位的元素，不会随着页面滚动条的滚动而滚动。

h2.pos_left
{
    position:relative;
    left:-20px;
}
h2.pos_right
{
    position:relative;
    left:20px;
}

相对定位会按照元素的原始位置对该元素进行移动。

样式 “left:-20px” 从元素的原始左侧位置减去 20 像素。

样式 “left:20px” 向元素的原始左侧位置增加 20 像素。

3、动画与特效

  1、变换（transform）

  在CSS3中提供了transform用于实现2D变换的属性，这个属性用于实现平移（transform），缩放（scale）、旋转（skew）和倾斜（rotate）的2D变换。

  以下表展示了各个属性的用法：

值 /函数 说明
none 表示无变换
translate(length,length) 表示实现2D平移，第一个参数对应水平方向，第二个对应Y轴，如果第二个参数伪类提供，则默认取第一个数的值
translateX(length) 表示在x轴（水平方向）上平移，参数表示移动移动的距离。
translateY(length) 表示在Y轴（垂直方向）上平移，参数表示移动的距离。
scale（number,number） 表示2D缩放，参数意义同上；
scaleX(number) 参数意义同上
scaleY(number) 参数意义同上
skew(angle,angle) 表示实现2D倾斜，参数意义同上，这里的参数指的是角度（deg）
skewX(angel) 参数意义同上
skewY(angle) 参数意义同上
rotate(angle) 表示2D旋转
matrix（6个参数） 代表一个基于矩阵变换的函数

  2、过渡(transition)

  在CSS3中也提供了用于过渡的属性transition，该属性可以控制某个HTML元素发生变化所用的时间，并以平滑方式发生改变，从而形成动画效果。

  1、指定参与过渡的属性：transition-property：all | none /<property>

  all：默认值，表示所有的过渡CSS属性

  none：表示不指定过渡的CSS属性

  2、指定过渡的参与时间：transition-duration：<time>(默认值0)

  3、指定过渡的延迟时间：transition-delay：<time>

  4、指定动画的类型：transition-timing-function：其中的属性值见下表：

属性值 说明
linear 线性过渡
ease 平滑过渡
ease-in 由慢到块
ease-out 由块到慢

  3、动画

  在CSS3中，实现动画需要经过两个步骤，一是定义关键帧，二是引用关键帧。格式如下：

  @keyframes name {keyframes-blocks}

  花括号中的代表的是动画在不同时段的样式，有以下两种：

  使用关键字from…to。例如：

@keyfeames opacityAnim{
    from {opacity:0}
    to {opacity:1}
}

  以上代码定义了一个名为opacityAnim的动画，用于实现从完全透明到完全不透明的动画效果。

  使用百分比定义关键帧的位置：

  例如：

@keyframes comple{
    0%{ opacity:0}
    20%{ opacity:1}
    50%{ transfrom:scale(0.8); }
    80%{ opacity:1}
    100%{ opacity:0}
}

  定义的动画用于实现从完全透明到完全不透明，再逐渐缩放80%，最后在由完全完全透明到完全不透明。

  4、动画属性

  常见的动画属性如下表：

属性 属性值 说明
animation 复合属性 用于指定对象所应用的动画特效
animation-name name 动画名
animation-duration time+单位(s) 动画持续时间
animation-timing-function 与值相关 指定过渡动画的类型
animation-delay time+单位(s) 动画延迟时间
animation-iteration-count number或infinite(无限循环) 动画循环次数
animation-direction 方向：normal默认值，alternate往返 是否方向运动
animation-play-state running（默认表示运动）paused（暂停） 动画状态
animation-fill-mode none（默认） 动画时间之外的状态

  设置动画时，可以将多个动画属性值写在一段代码中，也可以（通过上面的复合属性）写在一行当中。
