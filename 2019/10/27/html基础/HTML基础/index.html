一、HTML概述

1、什么是HTML

​ 超文本标记语言(Hyper Text Markup Language)，简称：HTML是一种用于创建网页的标准标签语言。 HTML语言是一种简易的文件交换标准，用于物理的文件结构，它旨在定义文件内的对象和描述文件的逻辑结构，而并不定义文件的显示，由于HTML5所描述的文件具有极高的适应性，因此它特别适合用于网页的出版环境。 同时它也是一个纯文本类型的语言，所编写出来的网页文件也是标准的纯文本文件。

2、HTML发展历程

  HTML的历史可以追溯到很久以前。1993年HTML首次以因特网草案的形式发布。20世纪90年代的人见证了HTML的大幅发展，从2.0版，到3.2版和4.0版，再到1999年的4.01版，一直到现在正逐步普及的HTML5。随着HTML的发展，W3C（万维网联盟）掌握了对HTML规范的控制权。

  在快速发布了HTML的前4个版本之后，业界普遍认为HTML已经无路可走了，对Web标准的焦点也开始转移到了XML和XHTML，HTML被放在次要位置。不过在此期间，HTML体现了顽强的生命力，主要的网站内容还是基于HTML的。为能支持新的Web应用，同时克服现有的缺点，HTML迫切需要添加新功能，制定新规范。

  2006年，W3C又重新介入HTML，并于2008年发布了HTML5的工作草案。2009年，XHTML2工作组停止工作。又过一年，也就到了现在。因为HTML5能解决非常实际的问题，所以在规范还没有具体订下来的情况下，各大浏览器厂家就已经按耐不住了，开始对旗下产品进行升级以支持HTML5的新功能。这样，得益于浏览器的实验性反馈，HTML5规范也得到了持续地完善，HTML5以这种方式迅速融入到了对Web平台的实质性改进中。

​

二、HTML文件的基本结构

​ HTML文件是由一系列的元素和标签组成的。元素是HTML的重要主成部分。

1、HTML的基本结构

（1）、标签

​ HTML的标签分为单独出现的标签（单独标签）和成对出现的标签（成对标签）两种。

​ 单独标签

​ 单独标签的格式为<元素名称>或<元素名称/>，其作用是在相应的位置插入元素，例如<br>标签，也可以写成<br/>,不过这里推荐大家使用前一种。

​ 成对标签

​ 大多数标签都是成对出现的，由首标签和尾标签组成，首标签的格式为<元素名称>，尾标签的格式为</元素名称>，其语法格式如下：

​ <元素名称>要控制的元素</元素名称>

​ 说明：在HTML中不区分大小写。每个HTML标签都可以设置一些属性，用来控制标签所建立的元素，这些属性位于首标签中，其语法格式如下：

​ <元素名称 属性1="值1" 属性2="值2">

​ 而在尾部则是这样的：</元素名称>

​ 因此，在HTML文件中某个元素的完整定义语法如下：

​ <元素名称 属性1="值1" 属性2="值2"......>元素资料</元素名称>

​ 注意：在HTML语法中，设置属性使用的 “ ” 可以省略。

（2）、元素

​ 当用一组HTML标签将一段文字包含在中间时，这段文字与包含这段文字的标签被称之为一个元素。比如<html></html>就可以称为一个html元素。

​ 在所有的HTML文档中，最外层的元素由<html>标签建立的，而在其中又包含了两个主要子元素，这两个子元素是由<head>标签与<body>标签所建立的，<head>标签所建立的元素内容为文件标题，而<body>标签所建立的元素内容为文件主题。

​ HTML文件结构：

<html>

<head>

<title>页面标题<title>

</head>

<body>

页面主体部分

</body>

<html>

2、HTML基本标签

​ 1、文件开始标签<html>

​ 在任何一个HTML文件里，最先出现的标签就是<html>，它用来表示该文件是用超文本标识语言编写的，<html>是成对出现的，首标签和尾标签分别位于文件的最前面和最后面。该标签不带任何属性。

​ 2、文件头部标签<head>

​ <head>是一个表示网页头部的标签，在标签放置一些HTML的文件信息，它不属于HTML文件的主体，它包含文件的标题、编码方式以及URL等信息。这些信息大部分是用于提供索引、辨认等的应用。

​ 3、文件标题标签<title>

​ 每个HTML文件都需要一个文件名称，在浏览器中，文件名称作为窗口名称显示在该窗口的最上方。并且<title>标签永远包含在<head>标签中。

​ 4、元信息标签<meta>

​ meta元素提供的信息是用户不可见的，它不显示在页面中，一般用来定义页面信息的名称、关键字、

作者等。在HTML中，<meta>标签不需要设置结束标签。

​ 5、页面的主体标签<body>

​ 以<body>标志它的开始，以</body>标志结束。在网页主体标签中常用的属性如下表：

​

属性 描述
text 设置页面文字颜色
bgcolor 设置页面背景颜色
background 设置页面背景图像
bgproperties 设置页面的背景图像为固定状态，不随页面的滚动而滚动
link 设置页面默认的链接颜色
alink 设置鼠标正在单击时链接的颜色
vlink 设定访问过后链接的颜色
topmargin 设置页面的上边距
leftmargin 设置页面的左边距

​ 6、页面的注释

​ 在HTML文件中注释分为三类，在<html></html>中注释、在CSS层叠样式表中注释，在JavaScript中注释。

​ 在<html></html>中注释，具体语法如下：

​ <!--注释的文字-->

​

​ 在CSS层叠样式表中注释，语法如下：

​ /*注释文字*/

​

​ 在 JavaScript中注释，语法如下：

​ 单行注释：//注释的文字

​ 多行注释：/* 注释的文字 */

三、HTML的全局属性

1、HTML的全局标准属性

  全局标准属性适用于大多数的标签，在HTML规范中，规定了以下8个全局标准属性：

  class：用于定义元素的类名。通常用于CSS样式的制作，偶尔也会使用到JavaScript。注意，class不能在以下元素中使用：<base>、<head>、<html>、<meta>、<param>、<script>、<style>、<title>。

  id：用于指定元素的唯一id。该属性的特点就是在整个HTML文档中具有唯一性，也就是说它的值不能重复。它也是用于CSS和JavaScript来指定元素的id添加样式、动作等。

  style：用于指定元素的行内样式。使用了该属性以后会覆盖全局的样式设定。

  title：用于指定元素的额外信息。通常指的是将鼠标移动到该元素时，提示的文本。

  accesskey：用于指定激活某个元素的快捷键。

  tabindex：用于指定元素在tab键下的次序。

  dir：同于指定元素中文本内容的文本方向。该属性只有两个值：ltr（left to right）从左到右，rtl（right to left）从右到左。默认为第一种。

  lang:用于指定元素内容的语言。

2、HTML的全局事件属性

  widow窗口事件：

  onload：页面加载完成以后触发的事件。

onunload：当用户卸载页面时触发的事件。（页面的卸载指的是：关闭页面、跳转其他页面、刷新页面等）

form表单事件：

onblur：当元素失去焦点时触发的事件。

onchange：当元素的值被修改以后触发的事件，

onfocus：当元素获得焦点时触发的事件。

onreset：当点击了重置按钮以后触发的事件。

onselect：当元素中文本被选中以后触发事件。

onsubmit：当提交表单时触发事件。

keyboard键盘事件：

onkeydown：当按下按键时触发的事件。

onkeyup：当按下键盘并松开时触发的事件。

onkeypress:当用户按下按键时触发，该事件不会对所有的按键都生效，不生效的按键有：alt、Ctrl、shift、esc键。

mouse鼠标事件：

onclick：当在元素上单击时触发的 事件。

ondblclick：当在元素上双击时触发的事件。

onmousedown：当在元素上按下鼠标时触发的事件。

onmouseup：当在元素上按下鼠标并释放时触发的事件。

onmouseover：当鼠标移动到元素时触发的事件。

onmousemove：当鼠标移动时触发的事件。

onmouseout：当鼠标移出元素时触发的事件。

media媒体事件：

onabort：当退出媒体播放时触发的事件。

onwaiting：当媒体已经停止播放并打算继续播放时触发的事件。

四、HTML的完整文档

  一个完整的HTML文档除了具备最基本的格式以外还应该包括以下内容：

  <! DOCTYPE> : 声明文档类型。

<base>:为页面上的所有链接规定默认地址或默认打开方式（target），该标签位于<head>标签当中。

<link>标签用于链接外部资源，通常用于链接CSS样式表。

<meta>标签提供了HTML的元数据，这些数据不会显示在页面，一般用于向浏览器传入数据或命令。

<meta>标签当中一共有两个属性，即name属性和http-equiv。

name属性：主要用于描述网页，如网页的关键字，叙述等。与之对应的属性值为content，content中的内容是对name属性的具体描述，通常用于搜索引擎抓取。name属性的语法格式如下：

<meta name = '参数' content = '具体描述'>

其中，name属性有以下几个参数：

  keywords,用于告诉浏览器网页的关键字等。

  description：用于告诉搜索引擎该网页的主要内容。

  robots：用于告诉搜索引擎网页是否需要索引，当name设置了该属性以后，content对应的值如下：

  all：默认参数，搜索引擎会索引此网页并且索引与该网页链接的其他网页。

  none：搜索引擎将忽略此网页。

  index：搜索引擎索引此网页。

  noindex：搜索引擎不索引此网页。

  follow：搜索引擎将索引网页及其该网页链接的其他网页。

  nofollow：搜索引擎不继续索引该网页链接的其他网页。

  author：用于标注网页的作者，通常跟后面会有文章。

  generator：用于标注该网页时用什么网页编写的。

  copyright：用于标注版权信息。

  revisit-after：设置爬虫访问时间。

  http-equiv：该属性与之前name属性类似，也是提供信息。对应的也是content。

  与之对应的参数为：

  content-type：用于设定网页字符集，通常使用“utf-8”。

  cache-control：用来告知浏览器如何缓存某个响应以及缓存的时间。它有以下几种参数：

  no-cache：先发送请求，与服务器确认资源是否被更改，如果未更改，则使用缓存。

  public：缓存所有响应，与max-age效果相同。

  private：只响应单个用户的缓存。

  max-age：表示当前请求开始，在多久内能被缓存和重用。不会去向服务器重新请求。

  expires：用于设定网页的到期时间。

  refresh：用于设定在规定时间内，自定刷新并转向设定的网址。

  set-cookie：用于设定网页的过期。
