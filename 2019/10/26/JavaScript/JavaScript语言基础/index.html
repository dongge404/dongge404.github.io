一、数据类型

​ JavaScript的数据类型分为基本数据类型和复合数据类型

1、数值型

​ 数值型（number）是JavaScript中最基本的数据类型。它和其他程序语言不同之处就在于它并不区分整型值和浮点型数值。包括我们之前学的python也是分整型和浮点型的。在JavaScript中所有的数值都是由浮点型表示的，它能表示的数值范围是1.7976931348623157e+308~5e-324。

​ 特殊值Infinity 在JavaScript中有一个特殊数值Infinity（无穷大），如果一个数值超过了JavaScript所能表示的最大范围（也就是上一条当中的1.7976931348623157e+308），JavaScript就会输出Infinity；如果一个数值超过了JavaScript所能表示的最小范围（也就是上一条当中的5e-324），JavaScript就会输出-Infinity

​ 在JavaScript中还有另外一个特殊值，那就是NaN(Not a Number)，即表示”非数字“。**如果在数学运算过程中出现了错误JavaScript就会返回NaN，它表示该数学运算结果是一个非数字。例如0除以0的输出结果就是一个非数字。

​

2、字符串型

​ 字符串（string）是由0个或多个字符组成的序列，它可以包含大小写字母、数字、标点符号或其他字符，同时也可以包含汉字。它是JavaScript用来表示文本的数据类型。程序中的字符串都是包含在单引号或者双引号中的，由单引号界定的字符串中可以含有双引号，同样，由双引号界定的字符串中也可含有单引号。注意：空字符串不包含任何字符，也不包含任何空格，用一对引号表示’‘’或‘’。

​ 有时候在字符串中使用引号会产生匹配混乱的问题。对于这种情况就要使用转义字符来进行转义，JavaScript的转义字符是“\”,通过添加转义字符可以在字符串中添加不可显示的特殊字符，或者防止引号出现混乱的问题。下表为JavaScript常用的一些转义字符：

3、布尔型

​ 1、布尔型

​ 布尔类型的数据只有两个值：一个是true，另一个是false。

​ 布尔值通常在JavaScript程序中用来作为比较后所得的结果。

​ 2、特殊数据类型

​ （1）、未定义值：undefined，表示变量还没有赋值

​ （2）、空值：null，表示一个空值，它不等同于空字符串（“”）或0。

​ （3）、null与undefined的区别：null表示一个变量被赋值为空值，另一个则表示还没有赋值，简单地说就是一个赋值，一个没有赋值。

二、常量和变量

1、常量

​ 常量是指在程序运行过程中保持不变的数据。这里就不多bb，反正大家都了解。

2、变量

​ 变量是指程序中一个已经命名的存储单元，它的主要作用就是为数据操作提供存放信息的容器。变量是相对于常言而言的。变量有两个基本特征：变量名和变量值。对于变量的使用必须明确以下几点：

（1）、变量的命名

​ 必须以字母或下划线开头。、

​ 变量名不能包含空格，加号或减号等符号

​ 严格区分大小写

​ 不能使用关键字

（2）、变量的声明

​ 在JavaScript中，变量的声明是由关键字“var”声明，其语法格式如下：

​ var variablename variablename是声明的变量名

（3）、变量的赋值

​ 在声明变量的同时也可以使用“=”对变量进行赋值。如下代码：

​ var lesson = "零基础学JavaScript" //声明变量同时进行初始化赋值

​ 另外也可以先声明变量后再对其赋值：

​ var lesson

​ lesson = "零基础学JavaScript"

​ 在JavaScript中可以不先声明变量而直接对其赋值。如下：

​ str = "这是一个未声明的变量"

​ 虽然这里可以使用此方法，但是这里博主建议先对变量进行声明是一种良好习惯，不仅能提高代码的可读性，还能及时发现错误。

​ 注意：如果只是声明了变量，未对其进行赋值，则值默认为undefined；也可以使用var语句重复声明同一个变量，也可以赋一个新的值：

​ var a;

​ var b = "你好，JavaScript;"

​ var b = "零基础学JavaScriptl;"

​ 注意：在JavaScript中的变量必须要先定义，才能使用

​ （4）、变量的类型

​ 变量的类型指变量的值所属的数据类型，可以是数值型、字符串、布尔型等。因为JavaScript是一种弱类型的程序语言，所以可以吧任意类型的数据赋值给变量。
