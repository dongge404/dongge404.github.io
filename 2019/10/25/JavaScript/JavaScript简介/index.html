一、JavaScript简述

  JavaScript是web页面中的一种脚本编程语言，也是一种通用的、跨平台、基于对象和事件驱动并具有安全性的脚本语言。它不需要进行编译，而是直接嵌入HTML页面中，把静态页面转变为支持用户交互并响应相应事件的动态页面。

  1、JavaScript的特点

​ 解释性：JavaScript不同于其他的编程语言，它是一种解释性的编程语言，它的源码不需要经过编译，可以直接在浏览器中运行时被解释。

​ 基于对象 : JavaScript是一种基于对象的语言，它能够运行自己创建的对象。

​ 事件驱动：JavaScript可以直接对用户的输入做出反应，无序经过web服务器，它对用户的响应，是以驱动的方式进行的，所谓事件驱动吗，就是指在主页中执行了某种操作所产生的动作，此动作就被称为“事件”。比如按下鼠标、移窗口、选择菜单等都可以视为事件

​ 跨平台：JavaScript语言依赖于浏览器本身，与操作环境无关，只要带有浏览器的计算机并且支持JavaScript就可以正确执行。

​ 安全性：可以保护数据的丢失。

  2、JavaScript的应用

​ 验证用户输入的内容；

​ 动画效果；

​ 窗口的应用；

​ 文字特效；

   二、web页面使用JavaScript的三种方法

  1、在HTML页面中直接嵌入JavaScript代码；

如果要直接在页面中嵌入JavaScript代码，可以使用以下标签：

<script>.......</script>

  这个标签既可以放在标签中，也可以放在标签中。

  该标签的属性及其说明如下表：、

  

属性 说明
language 设置使用的语言及其版本
source 设置外部脚本文件的位置
type 设置使用的脚本语言
defer 当页面加载完毕后执行的脚本语言

  注意：type属性在HTML4.0版本就已经取代了language属性

  1、language属性

<script language="JavaScript1.5">

  注意：如果不定义language属性，浏览器默认脚本语言为JavaScript1.0

  2、src属性

<script src="01.js">

  src属性用来指定外部脚本文件事物路径，其扩展名为.js。

  3、type属性

  type属性用来指定HTML文档使用脚本语言及其版本，自HTML4.0版本开始推荐使用type属性来代替language属性，type属性的使用格式如下：

<script type="text/JavaScript">

  4、defer属性

  defer属性的作用是当文档加载完毕以后再执行脚本语言。当脚本语言不需要立即运行时，可以设置defer属性，浏览器不必等待脚本语言加载，这样页面会加载的更快。defer属性到的适应格式如下：

<script defer>

  2、连接外部JavaScript文件

  如果代码比较复杂或是同一段代码可以被多个页面所使用，则可以将这些代码放置在一个单独的文件中，(保存文件的扩展名为.js)，然后在需要使用该代码的web页面中连接该JavaScript文件即可。在web页面中链接外部JavaScript文件语法格式如下：

<script type="text/JavaScript" src="JavaScript.js"></script>

  3、作为标记的属性值使用

  在JavaScript脚本程序中，有的JavaScript代码可能需要立即执行，而有的JavaScript代码可能需要单击某个链接或触发一些事件(如单击按钮)之后才会执行。

  （1）、通过“JavaScript:” 调用

  在HTML中，可以通过”JavaScript:”的方式来调用JavaScript的函数和方法,示例代码如下：

<a href="JavaScript:alert('您单击了这个超链接')">请点击这里`

  上述代码中通过使用“JavaScript:”的方式调用alert()方法，但是该方法并不是在浏览器解析到“JavaScript:”时就立即执行，而是在单击链接时执行。

  （2）、与事件结合使用

  JavaScript支持很多事件，而事件可以影响用户操作。比如鼠标左键、按下键盘或者移动鼠标等。与事件结合，可以调用执行JavaScript的函数和方法。实例代码如下：

<input type="button" value="单击按钮" onclick="alert('您单击了这个按钮')"/>

  三、JavaScript基本语法

  1、执行顺序

  JavaScript程序按照在HTML文件中出现的顺序逐行执行。如果需要在整个HTML文件中执行(如函数、全局变量等)，最好将JavaScript代码放在head标签中。

  2、大小写敏感

  JavaScript语言作为一门编程语言，它也就对大小写特别敏感。因此在输入语言的关键字、函数名、变量以及其他标识符时，都必须采用正确的大小写格式。

  3、空格与换行

  在JavaScript中会忽视程序中的空格、换行符合制表符。除非这些符号是字符串或者正则表达式中的一部分。另外JavaScript中的换行有“断句”的意思，即换行能判断一个语句是否已经结束。

  4、每行结尾的分号可有可无

  在JavaScript中，每个语句的结尾的分号 “ ；”可有可无，但是为了养成良好习惯，最好在每行语句的后面加上分号，以此增强代码的准确性。

  5、注释

  为程序添加注释有以下两种作用：

  （1）、可以解释某些程序的某些语句的作用和功能，是程序更容易理解，通常用于代码的解释说明；

  （2）、可以用注释来暂时屏蔽某些语句，使浏览器对其暂时忽略，等需要时再取消注释，通常用于代码的调试。

  在JavaScript中，提供了两种注释符号：

// 用于单行注释

/*......*/ 用于多行注释，它分为开始和结束两个部分
