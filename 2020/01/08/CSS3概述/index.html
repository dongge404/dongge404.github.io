一、CSS3

  1、CSS3中新增的选择器

  兄弟选择器

  在前面接触过一个相邻兄弟选择器，它是选择两个拥有相同父元素并相邻的子元素。而兄弟选择器则是选择第一个元素后面的所有元素，这些元素都有相同的父元素，后面的元素之间不一定要相邻，语法：元素1~元素2{property1:value1……}

  例子：

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
        <style type="text/css">

            h1~p{
                color: #FF0000;/*用于修改h1标签下的使用p标签*/
            }

        </style>
    </head>
    <body>
        <h1>标题1</h1>
        <p>文本1</p>
        <h2>标题2</h2>
        <p>文本2</p>
        <h3>标题3</h3>
        <p>文本3</p>
    </body>
</html>

  新增的属性选择器：

属性选择器 描述
E[attribute^=value] 选取指定值开头的属性值元素
E[attribute$=value] 选取指定值结尾的属性值元素
E[attribute*=value] 选取包含指定值的属性值元素

  新增的伪类选择器：

伪类名 含义
:root 选择文档的根元素，在HTML中都为<html>
:last-child 向元素添加样式，该元素是它父元素的最后一个子元素
:nth-child(n) 向元素添加样式，该元素是它父元素的第n个子元素
:only-child 向元素添加样式，该元素是它父元素的唯一子元素
:first-of-type 向元素添加样式，该元素是同级类型中第一个元素
:last-of-type 向元素添加样式，该元素是同级类型中的最后一个元素
:nth-of-type(n) 向元素添加样式，该元素是同级类型中第n个元素
:nth-last-of-type(n) 向元素添加样式，该元素是同级类型中倒数第n个元素
:only-of-type 向元素添加样式，该元素是同级类型中的唯一元素
:empty 向没有子元素（包括文本）的元素添加样式

  新增的伪元素选择器

伪元素名 含义
:enabled 向当前处于可用状态的元素添加样式（表单或者超链接）
:disabled 向当前处于不可用状态的元素添加样式（表单或者超链接）
:checked 向当前选中状态的元素添加样式
:not(selector) 向不是select元素的元素添加样式
:garget 向正在访问的锚点添加样式
::selection 向当前用户选取的内容所在的元素添加样式

  background-origin:

  为了兼容各大浏览器，我们需要写成这样的个格式“-webkit-background-origin”才可使用。这是因为浏览器的兼容问题所导致的，各大浏览器的前缀去下表所示：

前缀 浏览器 实例 说明
-ms- IE -ms-animation IEl浏览器专用属性
-moz- Firefox -moz-animation 火狐浏览器专用（基于Gecko内核）
-o- Opera -o-link Opera专用
-webkit- Chrome -webkit-background-origin 谷歌专用（基于webkit内核）

  该属性和前面的有点类似，它表示背景覆盖的起点，它们的值虽然相同，但是表示的含义不同，前者表示覆盖范围。后者表示覆盖起点。

  border-box：背景起点在边框左上角。

  padding-box：背景起点在内边距的左上角，默认值。

  content-box：背景起点在内容的左上角。

  4、CSS3盒模型新增的属性

  在CSS3中，新增了两个盒模型的新属性：圆角边框（border-radius）和阴影（box-shadow）。

  圆角边框细分四个属性，分别为：border-top-left-radius、border-top-right-radius、border-bottom-left-radius、border-bottom-right-radius。分别代表左上角、右上角、左下角和右下角的圆角边框。可以把以上的四条属性简写成一条属性，即border-radius。里面的值分别有1~4个。与前面的边距复制规则一样，但是不同的是只设定一个值时，代表的是四个方向的圆角边框。两个值时，第一个代表左上角与右下角的值，第二个代表右上角与左下角的值；三个值时，第一个代表左上角的值，第二个代表右上角与左下角的值，第三个代表右下角的值。比如以下代码：

#d1{
                width: 200px;
                height: 200px;
                margin: 20px auto;
                background-color: #0000FF;
                border-radius:50px;/*代表四个方向上的值*/
                border-radius:50px 70px;/*第一个为左上角与右下角值，第二个为右上角与左下角值*/
                border-radius:50px 70px 90px; /*第一个为左上角值，第二个为右上角与左下角值，第三个为右下角值。*/
                border-radius:50px 70px 90px 110px;/*分别代表四个方向的值*/
            }

  通常我们给圆角边框指定一个值，它表示水平半径和垂直半径的值，如果指定两个值则第一个表示水平半径、第二个表示垂直半径。这个时候需要斜杆设置值：

  border-radius：2em 1em 4em/0.5em 3em 斜杆前面的数值表示水平半径，斜杆后面的数值表示垂直半径，它等价于：

  border-top-left-radius：2em 0.5em

  border-top-right-radius：1em 3em

  border-bottom-right-radius：4em 0.5em

  border-bottom-left-radius：1em 3em

  【注意】在元素的高度和宽度相等时，我们的圆角边框数值超过宽度和高度的一半则变成一个圆，百分比值为50%：

#d1{
                width: 200px;
                height: 200px;
                margin: 20px auto;

                border-radius:50%;
                }
#d1{

  盒阴影（box-shadow）是盒模型的另一个新属性。它有以下六个属性值：

  h-shadow：水平方向偏移的距离，必选参数。

  v-shadow：垂直方向偏移的距离，必选参数。

  blur：模糊半径的距离，可选参数。

  spread：阴影额外增加的尺寸。负值为减少尺寸。可选参数。

  color：表示阴影颜色，可选参数

  inset：切换为内部阴影，可选参数。

  5、CSS3新增的变形动画属性

  在CSS3中新增了变形和动画属性，在这之前，这些动画都需要通过看JavaScript实现。

  变形属性

  CSS3提供了变形属性，分别是transform和transform-origin，可以让元素实现位移、旋转、缩放、倾斜等几种几何变换的操作。也可以将这些属性结合起来使用从而产生平滑的动画效果。

  transform：用户设置元素的变形，可以设置一个或多个变形函数。该属性提供了一下几种变换方式：

  translat(x,y):元素水平方向移动x，垂直方向移动y。其中y可以省略，表示只在水平方向上位移。

    translateX(x)：表示元素水平方向移动x

    translateY(y)：表示元素垂直方向移动y

  rotate(angle):表示元素顺时针旋转angle角度，angle通常以deg为单位。

  scale(x,y):元素水平方向缩放x，垂直方向缩放y。其中y可以省略，表示x与y同比例缩放。

    scaleX(x):表示元素水平方向缩放x

    scaleY(y):表示元素垂直方向缩放y

  skew(angleX,angleY)：表示元素沿着x轴倾斜angleX角度，沿着y轴倾斜angleY角度。

    skewX(angleX):表示元素沿着x轴倾斜angleX角度。

    skewY(angleY):表示元素沿着y轴倾斜angleY角度。

  matrix(a,b,c,d,x,y):将所有的2d函数组合在一起（平移、旋转、缩放、倾斜）

  transform-origin：表示元素旋转的中心点，默认值为50%,50%。第一个值表示中心点的水平位置，参数为：left、center、right、长度、百分比。第二个值表示中心点的垂直位置，参数为：top、center、bottom、长度、百分比。

  例子1：

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
        <style type="text/css">
            div{
                width: 100px;
                height: 100px;
                border: 1px solid;
                background-color: #FF0000;
            }
        </style>
    </head>
    <body>
        <div style="transform: skewX(30deg); transform-origin: left top; position: absolute;" >

        </div>
        <div style="position: absolute;background-color: transparent;">
            <p>沿着x轴的正方向倾斜，即y轴增加30度</p>
        </div>
        <div style="transform: skewY(30deg);transform-origin: left top; position: absolute; top: 200px;">

        </div>
        <div style="position: absolute; background-color: transparent; top: 200px;">
            <p>沿着y轴正方向倾斜，即再来x轴上增加30度</p>
        </div>
        <div style="transform: skew(30deg,30deg);transform-origin: left top; position: absolute;top: 400px; ">

        </div>
        <div style="position: absolute;background-color: transparent;top: 400px;">

        </div>
    </body>
</html>

  5、CSS3过渡属性（transition）

  所谓过渡就是从一个样式慢慢的变为另一个样式的过程，它有两个必选值，即过渡时间（transition-duration）以及过渡属性（transition-property），其他常见的属性如下表：

属性 含义 属性值
transition-delay 过渡延迟时间 time
transition-duration 过渡时间（必选），默认为0，如果不设置将不会产生任何结果 time
transition-property CSS过渡属性 none/all/CSS属性
transition-timing-function 时间曲线 linear/ease/ease-in/ease-out/ease-in-out/bezier(x1,y1,x2,y2)
transition 简写 以上四条属性

  6、CSS3动画属性（animation）

  所谓动画就是可以让元素自己完成一系列的动作（变换）。常见的属性如下：

属性 含义 属性值
@keyframes 定义动画的规则（关键帧） name + 时间 + CSS样式
animation-name 动画名称 自定义
animation-delay 动画延迟时间 time
animation-duration 动画时间 time
animation-timing-function 时间曲线 linear/ease/ease-in/ease-out/ease-in-out/bezier(x1,y1,x2,y2)
animation-iteration-count 动画播放次数 数字/infinit
animation-direction 动画反向播放 normal/alternate
animation-play-state 动画播放状态 paused/running
animation 简写属性 以上几条属性

  例子：

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">

        <style type="text/css">

                50% {top: 500px; left: 500px; background-color: yellow;}
                75% {top: 500px; left: 0px; background-color: green;}
                100% {top: 0px; left: 0px; background-color: red;}
            }
            div{
                width: 100px;
                height: 100px;
                background-color: red;
                position: relative;
                animation:turn 7s infinite;
                animation-direction: alternate;
            }    

        </style>
    </head>
