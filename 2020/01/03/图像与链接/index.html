图像与链接

1、图像

1、图像的基本格式

GIF格式

    此图片格式采用LZW压缩，以压缩相同色块来减少图像大小的，由于LZW压缩不会造成任何品质上的损失，而且效率也非常高，再加上GIF格式能在任何平台上都可以使用，所以现在成为一种主流。但是此格式只能处理256色。

JPEG格式

    对于照片之类全彩的图像，通常以JPEG格式保存，也就说明此格式的图片大多都超过256色，JPEG格式在压缩过程当中会造成图像数据的损失。

PNG格式

    这种格式是一种非破坏性的网页图像格式，它提供了将图像文件以最小的方式压缩却又不造成图像失真的技术。它不仅具备了以上两种格式的优点，而且还具备48-bit的色彩，更快地交错显示。

2、添加图像

  在HTML中插入图像使用<img>标签，（注意：这个标签没有结束标签）其语法格式如下：

<img src="图像文件的地址">

  src用来设置图像的文件地址，可以是相对地址，也可以是绝对地址。

  什么是绝对地址？什么是相对地址？

  绝对地址：指的是主页上的文件或目录在硬盘上的真正路径，例如D:\mr\5\5-1.png。使用绝对路径定位链接文件比较清晰，但是它有两个缺点：一是需要输入更多的内容，容易出错；二是如果该文件移动了，就要重新指定新的路径，比较麻烦。

  相对路径：相对路径则最适合网页内部文件引用，只要属于同一个网站，即使不在同一目录都可以使用。这种路径不受站点文件夹所处在服务器位置的影响，即使站点文件夹所处服务器地址发生改变，文件夹内部所有的文件地址都不会出现问题。

  相对路径的引用方法：

  • 如果要引用的文件位于该文件的同一目录下，则只需要输入链接文件的名字即可，例如 5-1.jpg

  • 如果要引用的文件位于该文件的下一级目录中，则需要输入目录名，然后添加一个“ / ”，再输入文件名，如mr/5-1.jpg

  • 如果要引用的文件位于文件的上一级目录中，则先输入“ ../ ”，然后在输入目录名、文件名。如 ../mr/5-1.jpg

3、图像大小与边框

调整图像大小

  在<img>标签中，通过width和height属性来调整图像的宽度和高度。语法格式如下：

<img src="图像文件地址" width="" height="">

  width：宽度，默认单位是像素，可以省略。

  height：高度，默认单位是像素，可以省略。

设置图像边框

  在默认情况下，我们插入的图像是没有边框的，但是可以通过border属性为图像添加边框，语法如下：

<img src="图像文件地址" border="">

4、图像的间距与对齐方式

调整图像的间距

  如果不使用<br>或<p>标签进行换行显示，那么插入的图像就会跟在文字后面。所以这里可以使用hspace和vspace调整图像与文字的距离，语法和前面一样。

  hspace：设置图像的水平距离，单位像素，可以省略；

  vspace：设置图像的垂直距离，单位像素，可以省略。

设置图像相对于文字基准线的对齐方式

  文字与图像的排列通过align参数来调整，其语法都雷同。

align取值 表示的含义
top 把图像的顶部与同行最高对齐，文本、图像都可以
middle 图像的中部和行的中部对齐
bottom 图像底部和同行文本底部对齐

替换文本与提示文字

  添加图像的提示文字—title，它表示当浏览到这张图像时，鼠标放在上面会出现提示文字。

  添加图像的替换文字—alt，当插入的图片由于某种原因无法显示时，就会使用该属性来代替。

2、链接

  链接(link)，也叫做超文本链接，通常叫超链接，它可以实现将文档中的文字或图像与另一个文档连接。常用的链接分为：文本链接、锚点链接、图像链接、图像热区链接、e-mail链接、空链接、JavaScript链接等。

1、文本链接

  在网页当中，文本链接是最常见的一种，语法格式如下：

<a href="" target="">链接名</a>

  href : 链接地址。

  target：打开新窗口的方式，它有以下四个属性值：

  • _blank:新建一个窗口打开。
  • _parent:在上一级窗口打开，通常用于分帧的框架页面。
  • _self:在同一个窗口打开。默认值。
  • _top:在浏览器的整个窗口打开。

2、锚点链接

  在一个HTML文档当中，可以分为很多小节，也就是可以在同一个页面当中可以跳转到不同位置，因此锚点链接就是实现在同一页面内之间的链接。锚点通常使用id属性或者name属性来设定，在a标签当中使用 “href” 属性添加“#锚点名”，比如以下代码：

<h1 id="home">首页</h1>
<a href="#home">回到首页</a>

  以上代码表示：h1标题中设置了id值，（这里也可以使用name属性），然后在此页面的某个地方设置了一个链接，该链接中的地址就是你设置的h1标题中id值，前面加上了一个“#”，这个时候当你点击了该链接以后就会自动跳转h1所在的位置。此时这个链接就实现了页面内部的链接，因此就称为锚点链接。

3、图像链接

  在给一幅图像设置超链接时，只要在<a></a>之间使用<img>标签插入图像就可以了。语法如下：

<a href="链接地址" target="窗口打开方式"><img src="图像地址"></a>

4、图像热区链接

  该类型的链接与上面的图片链接的区别在于：前者点击一张图片就会跳转到令一个页面，而后者点击一张图片的某个位置会跳转到一个页面，而当继续点击这张图片的另外一个位置时，会跳转到另一个页面上。总而言之，就是图片链接只会跳转到一个页面上，而图像热区链接则是跳转到一个或多个页面。

  在使用该标签时，就不再使用我们的<a>标签了，而是使用<map>标签，该标签用于定义映射图像的名称。在该标签当中使用<area>标签，在该标签中，有两个属性shape（定义形状）、coords（区域坐标）。如下表：

shape属性 解释 coords属性 解释
circle 圆形 x,y,r (x,y)表示圆心坐标，r表示的是半径
rect 矩形 x1,y1,;x2,y2 (x1,y1)为左上角坐标；(x2,y2)为右下角你坐标
poly 多边形 x1,y1;x2,y2;x3,y3….. (x1,y1),(x2,y2),(x3,y3)的是对应各个点的坐标

5、e-mai链接

  当我们设置了该类型的链接以后，浏览器会使用系统默认的e-mian程序。它的格式（href）是由“mailto”和e-mai地址两个部分组成的。比如<a href="mailto:2213568480@qq.com"></a>。

6、空链接

  该类型的链接是为指定目标地址的链接，一般情况使用较少。通常将href属性设置为JavaScript:viod(0)。有的时候回把它写成href=“”或者href=“#”。其实都是可以的，只不过后者具有“#top”的一个效果。

7、JavaScript链接

  当用户点击JavaScript链接时会触发JavaScript程序，比如下面代码：

<a href="javascript:alert('hello world!')"></a>
