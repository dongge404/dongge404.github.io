HTML表格与列表元素
HTML
发布日期:   2020-01-03
文章字数:   1.3k

1、HTML列表元素

  在HTML中，有以下几种列表：有序列表、无序列表和自定义列表。

  有序列表（Ordered list ）

  顾名思义，有序列表就是按照一定顺序进行排列的列表。在HTML当中，使用<ol>标签定义有序列表。用<li>标签定义列表中项，在<ol>标签当中，有一个type属性，该属性用来设置有序列表的项目符号，它的默认是数字（number）排序，还有其他的属性A（大写字母排序）、a（小写字母排序）、i（小写罗马排序）、I（大写罗马排序）。以下展示了几种不同类型的项目符的有序列表：

  无序列表（Unordered list ）

  参照上面讲的有序列表，无序列表就是不按照顺序排列的列表，使用<ul>定义，同样使用<li>标签定义列表项。虽然是无序列表，但是也有列表符。同样也通过type属性来设置，默认是一个黑点（disc）。还有其他两个值分别为：空心圆点（circle）、方形（square）。效果如下：

2、HTML表格元素

  在HTML中使用<table>标签来定义表格。每一个表格都是由若干行和列组成，使用<tr>（table row ）来定义表格行，一般在一个<table>标签中有几对<tr></tr>就说明该表格有几行。而使用<td>（Table data ）来定义表格中的数据，一般第一行或者第一列中的数据作为表头（table head），因此使用<th>标签定义。定义的表头与其他表格中的数据的区别就是表头字体默认加粗和居中的；而其他单元格数据不加粗，也不居中（左对齐）。

  当一个表格分为了头部、主体、底部时，可以使用<thead>、<tbody>、<tfoot>标签构造一个完整的表格。在表格中定义标题使用<caption>标签。

  <table>标签常用属性如下表：

属性 含义
border 表格的边框
width 表格的宽度
height 表格的高度
cellpadding 内边距
cellspace 外边距

  制作一个简单的表格代码如下：

<table>
    <tr>
         <th>姓名</th>
         <th>学号</th>
         <th>分数</th>
    </tr>

    <tr>
        <td>小明</td>
        <td>01</td>
        <td>69</td>
    </tr>
     <tr>
        <td>小红</td>
        <td>02</td>
        <td>78</td>
    </tr>
     <tr>
        <td>小梅</td>
        <td>03</td>
        <td>59</td>
    </tr>

  在<td>中有两个常用的属性：colspan（跨列）和rowspan（跨行），看下图：

  从上图可以看出，我们的蓝色两个单元格属于跨列（rowspan），而红色的两个单元格则是跨行（colspan）。也可以这样记：colspan跨列表示单元格水平方向跨越的列数，而rowspan则是跨行，表示垂直方向上跨越的行数。

  在<table>标签中还要掌握两个标签，即<col>和<colgroup>。<col>标签是为表格的一列或者多列定义属性。而<colgroup>则是用来对表格中的列进行组合，也就是说前者一般包括在后者当中。如下面代码：

<table>
<colgroup>
     <col style= "bgcolor = red">   <!--表示为表格的四列设置背景颜色样式-->
     <col style= "bgcolor = blue">
     <col style= "bgcolor = yellow">
     <col style= "bgcolor = green">
</colgroup>
</table>

  【注】：我们在制作表格的时候一定要切记我们的<tr>、<td>、<th>、<caption>、<col>、<colgroup>等所有关于表格内容的标签都要写在<table>标签内部。
