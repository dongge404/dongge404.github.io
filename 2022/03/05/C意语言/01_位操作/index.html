总所周知，计算机中最基本的处理单位是字节（Byte），一个字节等于八个比特。即1B = 8bit。

十进制和二进制

十进制我们已经很熟悉了，生活中最常用的就是十进制数字，比如18,19,20三个数字。我们知道计算机只会处理二进制的数据。因此学会十进制和二进制之间的相互转换也是非常重要的。

十进制转换二进制

短除法

对于这个方法，我们其实已经很熟悉了。就是对于任意一个十进制的数，我们都可通过短除法将其转换为二进制序列的数字。对于这个方法，只需要将被转换的数字除以2，得到商和余数，有余数在该被除数后面写0，没有余数写1.再对得到的商除以2，同前面的原理是一样的。最后会得到一个至上而下有0和1组成的序列，对这个序列进行从下到上写出来，即使这个十进制对应的二进制序列。

二进制转换十进制

位权展开法

二进制的每个数值都对应着一个位权，以一个字节为例，它有8个比特，每个比特对应着一个位权，他们分别是2^8、2^7、2^6、2^5、2^4、2^3、2^2、2^1、2^0。二进制序列的最高位一般都是在左边，从高到低依次往右。比如1101这个二进制序列，使用位权展开就可以得到：2^0 * 1 + 2^1 * 0 + 2^2 * 1 + 2^3 * 1 = 13。因此得到的十进制数就是13。

位运算

位逻辑与运算 &

位逻辑与运算要和逻辑与运算区分开，位与运算时二进制位之间的运算，逻辑与运算时表达式之间的运算。为了区分它们，分别使用& 和 &&代表位逻辑与运算和逻辑与运算。

位逻辑或运算 |

位逻辑或运算和逻辑或运算一样，只不过跟上面说的一样，只是操作对象的不同。

位逻辑异或 ^

位逻辑异或运算的宗旨为：相同为0，也就是比较的两个二进制位相同，都是1或者都是0，结果就是0，相异为1，就是说两个二进制位，一个为1，一个为0，结果就是1。

取反运算 ~

取反运算就是说将每个二进制位取反，1对应0,0对应1这样。

在上面的代码中，我们定义了一个printBinary的函数，这个函数接收一个无符号的1和字节的整型数据（这里采用无符号1个字节的char类型是因为我们用来计算一些简单的二进制数据，并且不带符号），由于任意一个数对2取余要么是0要么是1，因此将余数作为这个数对应的二进制序列是最合适的。其结果为0111011。那么我们都知道，这个函数现在的输出是按照从高到低的，但是我们需要将这个结构从低到高输出，因此可将这个算法做进一步的改进：

#include <stdio.h>
void printBinary(unsigned char dec)
&#123;
    char bites[8];
    int count = 0;
    unsigned int quotitent; //商
    unsigned int remainder; //余数
    while (dec > 0)
    &#123;
        //求余数
        remainder = dec % 2;
        //求商
        quotitent = dec / 2;

        dec = quotitent;  //将计算出来的商作为新的被除数

        bites[count] = remainder;  //将得出的余数放进数组当中
        count++;
    &#125;
    //逆序输出数组中的数据
    for (int i = count - 1; i >= 0; i--)
    &#123;
        printf("%d", bites[i]);
    &#125;

&#125;
int main()
&#123;
    printBinary(110);
    return 0;
&#125;

上面的代码中，我们定义了一个数组，用于保存余数的值，还定义了count变量，用于记录余数的个数，每次循环都将得到的余数放入数组当中，最后通过逆序，就是从后往前输出数组中的余数，就可以得到该数值对应的二进制序列，调用函数，输入的110，结果为1101110。当然，这个算法还有一定的缺陷，那就是当你输入0时，不会有任何输出，这是因为，该值不满足循环条件，自然不会参与计算，由于数组只是声明，并未对其初始化，因此循环输出数组，也是没有任何数据的。我们可以再做进一步的改进：

左移运算和右移运算

左移运算 <<

左移运算是将数据对象中的二进制位向左移动指定的位数，高位多出来的会被移除，低位则补充0

右移运算 >>

右移运算是将数据对象的二进制位向右移动指定的位数，对弈无符号数，高位补充0，对于有符号数，填充0或者1，由编译器决定的。这里暂时只探讨无符号的情况。

当然，移动几位取决于该运算符后面的数字，这个数字不应该太大或者太小，一般都是在1~3之间。
